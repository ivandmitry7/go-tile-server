API Documentation

REST API reference for integrating with the Tile Server.

Tile URL Format

Serve downloaded map tiles via standard XYZ URL pattern:

GET /tiles/{z}/{x}/{y}.png
ParamDescription
zZoom level (0-20)
xTile column
yTile row
CORS: All tile responses include Access-Control-Allow-Origin: * header, so tiles can be used from any domain.

Leaflet.js

L.tileLayer('http://YOUR_HOST:8080/tiles/{z}/{x}/{y}.png', {
    maxZoom: 20,
    attribution: 'Offline Tiles'
}).addTo(map);

OpenLayers

new ol.layer.Tile({
    source: new ol.source.XYZ({
        url: 'http://YOUR_HOST:8080/tiles/{z}/{x}/{y}.png',
        maxZoom: 20
    })
});

MapLibre GL

map.addSource('offline-tiles', {
    type: 'raster',
    tiles: ['http://YOUR_HOST:8080/tiles/{z}/{x}/{y}.png'],
    tileSize: 256,
    maxzoom: 20
});

POST /api/download

Start downloading tiles for a polygon area.

Request Body

FieldTypeDescription
polygon[{lat, lng}]Polygon vertices (min 3 points)
holes[[{lat, lng}]]Exclusion zones (optional)
zoom_minintMinimum zoom level (0-20)
zoom_maxintMaximum zoom level (0-20)

Response

{
    "task_id": "task-1",
    "total": 12345,
    "status": "running"
}

Validation

  • polygon must have at least 3 points
  • zoom_min and zoom_max must be 0-20
  • zoom_min must be ≤ zoom_max

Examples

curl
JavaScript
curl -X POST http://localhost:8080/api/download \
  -H "Content-Type: application/json" \
  -d '{
    "polygon": [
      {"lat": 16.05, "lng": 108.18},
      {"lat": 16.05, "lng": 108.22},
      {"lat": 16.02, "lng": 108.22},
      {"lat": 16.02, "lng": 108.18}
    ],
    "holes": [],
    "zoom_min": 10,
    "zoom_max": 15
  }'
const resp = await fetch('/api/download', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
        polygon: [
            { lat: 16.05, lng: 108.18 },
            { lat: 16.05, lng: 108.22 },
            { lat: 16.02, lng: 108.22 },
            { lat: 16.02, lng: 108.18 }
        ],
        holes: [],
        zoom_min: 10,
        zoom_max: 15
    })
});
const data = await resp.json();
console.log('Task ID:', data.task_id);

GET /api/progress?task_id=X

Server-Sent Events (SSE) stream for real-time download progress.

Query Parameters

ParamTypeDescription
task_idstringTask ID from download response

SSE Data Format

{
    "done": 1500,
    "total": 5000,
    "errors": 2,
    "status": "running",
    "current_zoom": 13,
    "zoom_min": 10,
    "zoom_max": 15
}

Status Values

StatusDescription
runningDownload in progress
completedAll tiles downloaded
cancelledUser cancelled
errorDownload failed

JavaScript Example

const es = new EventSource('/api/progress?task_id=' + taskId);
es.onmessage = function(e) {
    const data = JSON.parse(e.data);
    const pct = Math.round(data.done / data.total * 100);
    console.log(`Progress: ${pct}% (${data.done}/${data.total})`);
    if (data.status === 'completed' || data.status === 'cancelled') {
        es.close();
    }
};

GET /api/tasks

Response

        "ID": "task-1",

        "Total": 5000,

        "ZoomMax": 15
    }
]

Examples

curl
JavaScript
curl http://localhost:8080/api/tasks
const tasks = await fetch('/api/tasks').then(r => r.json());
console.log(tasks);

Get a specific project configuration.

Response

{
    "name": "My Project",
    "polygon": [{"lat": 16.05, "lng": 108.18}, ...],
    "holes": [],
    "annotations": [
        {
            "type": "marker",
            "name": "Port",
            "color": "#e94560",
            "icon": "⚓",
            "latlngs": {"lat": 16.05, "lng": 108.20}
        },
        {
            "type": "polyline",
            "name": "Route",
            "color": "#3498db",
            "latlngs": [{"lat": 16.05, "lng": 108.18}, ...]
        }
    ],
    "zoom_min": 10,
    "zoom_max": 15,
    "map_center": {"lat": 16.04, "lng": 108.20},
    "map_zoom": 12

Examples

curl
JavaScript
curl "http://localhost:8080/api/drawings?name=My%20Project"
const config = await fetch('/api/drawings?name=' + encodeURIComponent('My Project'))
    .then(r => r.json());

GET /api/tile-size

Get tile storage information (total size and file count).

Response

{
    "size": 104857600,
    "files": 25000
}
FieldTypeDescription
sizeint64Total storage size in bytes
filesint64Number of tile files

Examples

curl
JavaScript
curl http://localhost:8080/api/tile-size
const info = await fetch('/api/tile-size').then(r => r.json());
const mb = (info.size / 1024 / 1024).toFixed(2);
console.log(`Storage: ${mb} MB (${info.files} files)`);
